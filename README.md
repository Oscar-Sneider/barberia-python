# Barbería Python

Programa en Python para calcular pagos y propinas en una barbería.

## Funcionalidades
- Solicita el nombre del cliente
- Calcula si el pago es suficiente
- Detecta propina
- Maneja errores de entrada
- Permite atender varios clientes

## Cómo usarlo
Ejecuta el archivo `barberia.py` desde la terminal o Thonny y sigue las instrucciones.
